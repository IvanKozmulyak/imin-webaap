# IMIN Organizer Commission Model

## Overview
Revenue model for charging event organizers a per-registration fee, positioning IMIN as a ticket-sales multiplier.

## The Business Case

IMIN provides organizers with:
- **+20% ticket sales** — solo attendees who otherwise wouldn't attend
- **Better attendee experience** — people arrive with pre-matched squads
- **Higher retention** — organized squads return more often  
- **Zero integration burden** — IMIN handles all matching

## Pricing Models

### Option A: Per-Ticket Commission (Recommended)
**€1.50 per matched attendee** (sold through IMIN)

- Simple, transparent pricing
- Organizers only pay for value delivered
- Scales with event size
- Example: 100 matched attendees = €150

### Option B: Flat Monthly Fee
**€500/month** for unlimited events

- Better for high-volume organizers (10+ events/month)
- Predictable costs for budgeting
- Encourages more event listings

### Option C: Hybrid Model
**€0.75 per ticket + €250/month**

- Lower per-ticket rate with monthly minimum
- Mid-ground between A and B

### Option D: Revenue Share
**5-10% of ticket price**

- Tied directly to ticket value
- Premium events pay more
- More complex to track

## Recommended Tiers

| Organizer Type | Model | Price Point |
|----------------|-------|-------------|
| Small (1-3 events/mo) | Per-ticket | €1.50/ticket |
| Medium (4-10 events/mo) | Hybrid | €0.75/ticket + €100/mo |
| Large (10+ events/mo) | Flat fee | €500/mo |
| Enterprise | Custom | Negotiation |

## Implementation Notes

### Technical Requirements
- Track matched attendees per event
- Generate monthly invoices
- Payment integration (Stripe, bank transfer)
- API for organizer dashboard

### Value Demonstration
- Show "incremental tickets" attributed to IMIN
- Provide post-event analytics
- Demonstrate squad satisfaction scores

### Early Adopter Strategy
- First 10 organizers: 50% discount (€0.75/ticket)
- Pilot program with 2-3 recurring events
- Build case studies before full pricing

## FAQ for Organizers

**Q: What if no one signs up through IMIN?**
A: No fee — you only pay for matched attendees.

**Q: Can I track how many IMIN users attend?**
A: Yes, dashboard shows matched vs. organic attendees.

**Q: Is this mandatory?**
A: Optional. Organizers can list events for free but won't get squad matching.
