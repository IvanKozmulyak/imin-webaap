## How to Use

### Option 1: Screenshot (Easiest)
1. Open `organizer-assets.html` in a browser
2. Toggle light/dark theme if needed
3. Screenshot each asset at your desired resolution
4. Post to your social channels

### Option 2: Customize & Generate
Edit the HTML directly to:
- Change colors to match your brand
- Update event names
- Add your specific event link
